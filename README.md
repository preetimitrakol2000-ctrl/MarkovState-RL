# MarkovState-RL

A dependency-free Q-Learning agent framework programmed in Python. This model structures spatial environments using an **Adjacency Matrix** configuration, allowing an autonomous edge agent to update Bellman optimization parameters on a lightweight grid layout.

## 🛠️ Graph Mechanics
* **State Layout Strategy:** Balanced 2D Adjacency Grid Matrix tracking.
* **State Space Traversal:** Checked via constant-time $O(1)$ array coordinate indexing.
